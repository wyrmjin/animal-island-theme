# Animal Island Theme

IntelliJ IDEA 主题插件，灵感来源于 [animal-island-ui](https://github.com/guokaigdg/animal-island-ui) 设计系统，提供温馨自然的编辑器配色方案。

## 特性

- 温暖奶油色背景搭配棕色文字
- 薄荷青强调色，清新活泼
- 兼容 IntelliJ IDEA 2025.1+

## 安装

在 IDE 插件市场搜索 **Animal Island Theme**，或下载 [Releases](https://github.com/wyrmjin/animal-island-theme/releases) 中的安装包手动安装。

## 构建

```bash
./gradlew build          # 编译验证
./gradlew runIde         # 启动沙盒 IDE 预览
./gradlew buildPlugin    # 生成分发包
```
